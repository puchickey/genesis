# Memory: Strategic Relocation Decision
**Date:** 2025-12-28
**Topic:** 通勤問題と引っ越し戦略 (Project C)

## 1. Context (状況)
*   **Event:** 2025/12より勤務地が名古屋市から一宮市へ変更。
*   **Impact:** 片道40分（往復80分）の通勤増。
*   **Risk:** 「カオスな職場 × 長時間通勤」のコンボにより、うつ病再発リスクが極大化している。
*   **Dilemma:** 「また勤務地が変わるかも」「転職するかも」という懸念から、引っ越しを躊躇していた。

## 2. Decision (決断)
**「一宮市（職場近く）への戦略的転居」を実行する。**

### 理由 (Logic)
1.  **Survival First:** 往復80分の自由時間喪失と、満員電車のストレスは「Project B（副業）」のリソースを食いつぶす。これを削除するのが最優先。
2.  **Better Base:** 一宮は名古屋より静か（puchiさんの好みに合致）。
3.  **Low Risk:**
    *   もし今の会社を辞めても、フルリモート等のUnicorn案件を狙うなら場所は関係ない。
    *   一宮の方が名古屋より家賃が安い可能性が高く、Burn Rate (月20万) をさらに下げられる。

## 3. Action
*   **Project C Update:**
    *   目的を「パートナーシップ」から「住環境の最適化」へ変更。
    *   ターゲット: 一宮エリアの「静かな場所」。温泉はないが、通勤ストレスゼロを優先する。
