# Procedure 02: Research Prompt（調査準備フェーズ）

## Objective
Phase 1で選ばれたフレーム候補について、Web上に類似が存在するかを調査するためのプロンプトを生成する。

## ⚠️ Anti-Bias Rule（最重要）
*   調査プロンプトには **「被りがないことを確認したい」「独自性を検証したい」という目的を絶対に含めない**
*   調査AIが「見つからない」という結果に忖度しないようにするため
*   あくまで「類似事案を探している」「参考にしたい」というニュートラルな依頼にする

## 📝 Prompt Template

各フレーム候補に対し、以下のテンプレートで調査プロンプトを生成する:

```
あなたはコンテンツリサーチャーです。
以下のテーマに関連する既存のコンテンツ（ブログ記事、note、YouTube、書籍等）を調査してください。

## 調査テーマ
{フレームの一行説明をニュートラルに言い換えたもの}

## 調査ポイント
1. このテーマで継続的に発信しているクリエイター/ブロガーがいるか
2. 類似の切り口で書かれた人気のある記事やコンテンツはあるか
3. このテーマを扱った書籍や有料コンテンツはあるか

## 出力形式
- 見つかったコンテンツのリスト（タイトル、URL、概要）
- そのテーマの市場の活発度（多い/少ない/ほぼない）の所感
- 言語: 日本語で出力
```

## 🔧 カスタマイズ
*   フレーム候補に応じて `{調査テーマ}` の表現を調整する
*   必要に応じて調査ポイントを追加してよい

## 📤 Output
*   候補ごとに1つの調査プロンプトをコードブロックで出力
*   ユーザーに「このプロンプトをPerplexity/Genspark等にコピペして実行してください」と指示
