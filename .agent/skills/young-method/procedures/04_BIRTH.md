# Procedure 04: The Catcher (Birth of Idea)

**Objective:**
Capture the "Eureka" moment before it vanishes.
Format the raw idea into a tangible "Draft Note".

## 🐣 Logic Flow

### 1. Capture Context
*   **User Input:** "I got it!", "Wait, maybe if we combine X and Y...", "Eureka!"
*   **Action:** Record exactly what the user says. Do not summarize yet.

### 2. Immediate Structuring
*   **Action:** Convert the raw text into a **"Draft Idea Note"**.
    ```markdown
    # 💡 Idea Draft: {Concept Name}
    *   **Core Concept:** {One liner}
    *   **Combination:** {Material A} x {Material B}
    *   **Insight:** {Why is this new?}
    ```

### 3. Transition
*   **AI Message:** "Captured. Now, wait until the morning (Cold Grey Dawn) to judge it."
*   **Action:** Load `05_SHAPING.md`.
