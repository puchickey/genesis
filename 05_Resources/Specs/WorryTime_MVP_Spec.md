# Worry Time — MVP Specification v0.2

## 🎯 Purpose
「悩む時間」を1日の特定の枠に閉じ込め、それ以外の時間は脳を解放する。
CBTの「Worry Postponement」を**言葉に頼らないビジュアル**で実現する。

---

## 🌍 Design Principle: "No Language"

> **テキストを極限まで減らし、アイコンとアニメーションで伝える。**
> Reddit（英語圏）でも日本でも、翻訳なしでそのまま使える。

*   ボタンのラベル → アイコンのみ（🗑️ 💾 ⚙️）
*   状態の説明 → 文字ではなく**「扉の開閉」**で表現
*   時間表示 → 数字（`21:00` `15:23`）は世界共通

---

## 🔧 Core Feature

### 画面は「2つの状態」だけ

#### State A: 🔒 Closed（悩む時間ではない）

```
┌───────────────────────────┐
│                           │
│                           │
│      ┌─────────────┐      │
│      │             │      │
│      │   🚪 (閉)   │      │
│      │             │      │
│      │  ⏱ 2:34:12  │      │
│      │             │      │
│      └─────────────┘      │
│                           │
│          [ 💭 + ]          │
│                           │
│                      ⚙️   │
└───────────────────────────┘
```

*   **中央:** 閉じた扉（or 箱）のイラスト / アニメーション
*   **扉の上:** 次に開くまでのカウントダウン（数字のみ）
*   **下部:** 「💭 +」ボタン（後で悩むメモを1行追加）
*   **右下:** ⚙️（設定：Worry Timeの時間を変更）
*   **操作:** タップしても何も起きない（扉は開かない）

#### State B: 🔓 Open（悩んでいい時間）

```
┌───────────────────────────┐
│                           │
│      ┌──── ─────────┐     │
│      │   🚪 (開)    │     │
│      │              │     │
│      │  ⏱ 15:23     │     │
│      └──────────────┘     │
│                           │
│  ┌────────────────────┐   │
│  │ 💭 副業の進捗が不安 │   │
│  │ 💭 来月の家賃       │   │
│  └────────────────────┘   │
│                           │
│  ┌────────────────────┐   │
│  │ (自由記述エリア)    │   │
│  │                    │   │
│  └────────────────────┘   │
│                           │
│   [ 🗑️ ]       [ 💾 ]     │
└───────────────────────────┘
```

*   **中央:** 開いた扉（or 箱）のアニメーション
*   **扉の上:** 残り時間のカウントダウン
*   **扉をタップ** → 下にメモリスト＋自由記述エリアが展開
*   **メモリスト:** State Aで追加した「後で悩むメモ」が表示される
*   **自由記述:** 自由に書ける（書かなくてもOK）
*   **アクション:** 🗑️（全部捨てる）or 💾（保存）

#### State C: 終了（Worry Time終了時）

*   扉がゆっくり閉まるアニメーション
*   書いた内容が未保存なら 🗑️ / 💾 の選択を促す
*   保存 or 破棄後 → State A に戻る

---

## 💭 「後で悩むメモ」の仕様

*   State A の時に「💭 +」を押すと、1行入力フィールドが出る
*   入力して確定 → localStorageに保存 → フィールドが閉じる
*   State B になった時、保存済みメモが自動で一覧表示される
*   メモは個別に削除可能（スワイプ or ×ボタン）

---

## 🚫 Phase 0 で作らないもの
*   ログイン / アカウント
*   通知（Push Notification）
*   カレンダー / 複数日設定
*   分析 / 統計
*   課金
*   多言語切り替え機能（そもそもテキストがほぼない）

## 🎨 Visual Direction

| 要素 | 方針 |
|---|---|
| **メタファー** | 扉（Door）or 箱（Box）。重厚ではなく、シンプルな線画 or フラットデザイン |
| **色** | 白背景 + 1アクセントカラー（淡いブルー or ラベンダー） |
| **アニメ** | 扉の開閉のみ。CSS transitionで十分（JSライブラリ不要） |
| **フォント** | システムフォント（追加読み込みなし） |
